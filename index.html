Cara Menggunakan Aplikasi

Selamat datang! Berikut panduan singkat untuk menggunakan fitur-fitur di halaman ini:

  • Nama Sekolah: Isi nama sekolah yang akan muncul di kop surat.
  • Nama / Kelas / Jenis Izin / Tanggal: Isi data siswa dan pilih jenis izin.
  • Catatan: Tambahkan keterangan tambahan (opsional).
  • Tanda Tangan: Gambar tanda tangan di kotak, atau tekan "Upload" untuk memakai gambar, lalu tekan "Gunakan Tanda Tangan" untuk memasukkannya ke preview.
  • Preview: Tekan "Tampilkan Preview" untuk melihat surat sebelum mendownload.
  • Download PDF: Tekan "Download PDF" untuk menyimpan surat (tanda tangan akan ikut tersertakan).
  • Mode Gelap: Gunakan tombol mode gelap di pojok untuk mengganti tema; pilihan tersimpan otomatis.
